# 💊 Smart Medicine Reminder Web App

A modern and user-friendly **Medicine Reminder Web Application** built using Flask.  
This app helps users manage their daily medicine schedule and provides a clean interface for tracking reminders.

---

## 🚀 Features

- ➕ Add medicines with time
- 📋 View all scheduled medicines
- ❌ Delete medicines easily
- 💾 Data persistence using JSON
- 🎨 Modern UI with glassmorphism design
- 🔔 Local notification system (Python script)

---

## 🛠️ Tech Stack

- **Python (Flask)** – Backend
- **HTML + CSS** – Frontend
- **JSON** – Data storage
- **Plyer** – Desktop notifications
